# ros2_control_demo_example_18

This example demonstrates how an ONNX policy can drive the Berkeley Humanoid Lite Biped robot to follow a command of walk forward at 0.5 m/s. This This also demonstrates observation obtained from interfaces_state_broadcasters. Design documentation: [doc/design.rst](doc/design.rst)

Find the full documentation in [doc/userdoc.rst](doc/userdoc.rst) or on [control.ros.org](https://control.ros.org/master/doc/ros2_control_demos/example_18/doc/userdoc.html).
