# ros2_control_demo_example_17

   *DiffBot*, or ''Differential Mobile Robot'', is a simple mobile base with differential drive. This example shows how to control a custom differential drive robot (diffbot) using ``gz_ros2_control::GazeboSimROS2ControlPlugin`` and Gazebo built-in ``gz::sim::systems::DiffDrive`` plugin in Rubicon world.

Find the documentation in [doc/userdoc.rst](doc/userdoc.rst) or on [control.ros.org](https://control.ros.org/master/doc/ros2_control_demos/example_17/doc/userdoc.html).
